## Sum Element

Given an array of integers, return the sum of all elements at a given index.

#### Example 1

```
Input: arr = [1,2,3,4,5]
Output: 15
```
Explanation: The sum of all elements in the array is 1 + 2 + 3 + 4 + 5 = 15.



#### Example 2

```
Input: arr = [1,1,1,4,5]
Output: 12
```

Explanation:
The sum of all elements in the array is 1 + 1 + 1 + 4 + 5 = 12.
